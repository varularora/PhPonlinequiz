# PhPonlinequiz

The BRAINFREEZE is a website for to take online test in an efficient manner and no time wasting for checking the paper. 
The main objective of BRAINFREEZE website is to efficiently evaluate the candidate thoroughly through a fully automated system that not only saves lot of time but also gives fast results. For students they give papers according to their convenience and time and there is no need of using extra thing like paper, pen etc. This can be used in educational institutions as well as in corporate world.
